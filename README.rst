cheb++
======

**cheb++** is a C++ library which implements a set of functions dealing with Chebyshev polynomials.

----------

2. Download
^^^^^^^^^^^

3. Install
^^^^^^^^^^

Dependency

* FFTW 3.3+
* Armadillo 4.300+

Mac OS x86_64
Add "-framework Accelerate" to CXXFLAGS in the Makefile
